# osa

> **흩어진 하루를 하나의 리듬으로**

---

## 프로젝트 정체성

**osa**는 영화감독이자 시나리오 작가, AI 영상 제작자, 로컬 비즈니스 마케터로 살아가는 1인 창작자의 하루를 하나의 리듬으로 정돈하는 개인 운영 시스템(Personal OS)입니다.

단순한 할 일 앱이 아닙니다. 몸·정신·창작·현실 업무가 균형 잡힌 삶을 위한 디지털 인프라입니다.

---

## 핵심 철학

- 하루를 바쁘게 보내는 것이 아니라, **예술가의 리듬으로 정돈**하는 것
- 매일 하나의 문장과 하나의 장면을 남기는 것
- 지혜롭고 깊은 사람이 되어가는 과정을 기록하는 것

---

## 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | React 19 + TypeScript |
| Styling | Tailwind CSS 4 |
| UI Components | shadcn/ui |
| Build | Vite |
| Package Manager | pnpm |

---

## 핵심 원칙 파일

- `CLAUDE.md` — AI 협업 원칙 및 코드 작성 기준
- `design_tokens.md` — 디자인 시스템 토큰 정의

---

## 모드 전환 명령어

| 명령어 | 역할 |
|--------|------|
| `/plan` | 기획 검토 및 로드맵 설계 |
| `/code` | 코드 리뷰 및 기술 구현 |
| `/ux` | UX 비평 및 인터페이스 개선 |
| `/day` | 하루 마무리 코칭 |
| `/copy` | 랜딩 페이지 카피라이팅 |
